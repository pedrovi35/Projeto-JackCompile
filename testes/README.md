# Arquivos de Teste — Square Dance

Estes arquivos são os testes oficiais do **capítulo 10** do nand2tetris, usados para validar o analisador léxico.

## Como usar

Com o projeto compilado, execute:

```bash
java JackAnalyzer testes\Square\
```

Serão gerados:
- `testes\Square\MainT.xml`
- `testes\Square\SquareT.xml`
- `testes\Square\SquareGameT.xml`

Compare os arquivos gerados com os arquivos `*T.xml` oficiais disponíveis no kit do curso em `nand2tetris/projects/10/Square/`.
